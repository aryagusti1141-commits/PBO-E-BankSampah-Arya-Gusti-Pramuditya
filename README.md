# PBO-E-BankSampah-Arya-Gusti-Pramuditya

## Deskripsi Proyek
Proyek ini bertujuan menciptakan sistem digital untuk pengelolaan bank sampah di sebuah komunitas. Sistem ini dirancang untuk menggantikan pencatatan manual agar identitas nasabah, nilai ekonomi jenis sampah, dan setiap setoran transaksi tercatat secara akurat dan efisien.
